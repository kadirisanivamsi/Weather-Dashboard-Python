# 🌦️ Weather Dashboard

A Python command-line application that fetches live weather information using the OpenWeather API.

## Features

- Search weather by city
- View temperature
- View humidity
- View pressure
- View wind speed
- Save search history
- Menu-driven application

## Technologies Used

- Python
- Requests
- JSON
- OpenWeather API

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
